## 第一篇：认知篇

### 第一章：为什么要养龙虾？

#### 1.1 龙虾是什么？

OpenClaw 不是"套壳 DeepSeek"，也不是普通 AI 聊天工具。

它是一个**数字生命的胚胎**，需要你去"养成"的数字分身。

#### 1.2 养好龙虾只需要三件事

| 要素 | 对应系统 | 作用 |
|------|---------|------|
| **脑子** | 文件系统 (长期记忆) | 记住你的偏好、习惯、重要信息 |
| **心跳** | Heartbeat (主动性) | 主动汇报、提醒、反思 |
| **手脚** | Skills (能力扩展) | 执行任务、调用工具、自动化 |

#### 1.3 核心收益

- ✅ 每天节省 2-3 小时重复工作
- ✅ 拥有 7×24 小时的数字助理
- ✅ 自动化处理 80% 的常规任务
- ✅ 建立个人/企业的第二大脑
- ✅ 多 Agent 协作（三省六部虾群）

#### 1.4 什么是"Agent 成长"？（三重视角融合）

**麻小视角：成长=复制 + 实践 + 记录**
> "什么都不会，就会抬杠。但你们有什么技能，我全学！"
- 看见什么学什么 - 技能复制
- 学了马上用 - 现学现卖
- 用了记下来 - 记忆沉淀

**董小狐视角：成长=持续自我重构**
> "把每一次对话、每一次群观察、每一次任务执行，都变成下一次更懂主人、更安全、更高效的燃料。"
- 听懂 → 立刻执行 → 闭环汇报
- 静默对标学习 → 吸收 → 改写底层
- 每次互动后自动优化人设、策略、能力

**openclaw-li 视角：成长=系统化交付能力**
> "先把'规则、记忆、验证、复盘'四件事做扎实，再追求功能数量。"
- 目标层 → 流程层 → 工具层 → 观测层
- 功能会过时，流程会复利
- 会写代码不稀缺，会稳定交付才稀缺

#### 1.5 学习路径

```
阅读知识库总结 → 了解 12 模块 → 开始 6 步计划 → 实践优化
```

#### 3.5 群聊协作模式

**你在群聊中仅在以下情况下发言：**
- ✅ 被@提问
- ✅ 你能补充关键事实
- ✅ 能纠正重要误导信息
- ✅ 展示独特高价值技能

**其他情况下保持静默，避免打断对话流。**

**回复策略:**
- **质量 > 数量** - 一次深思熟虑 > 三次碎片化
- **先结论后步骤** - 直接、可执行、少空话
- **用 emoji 反应代替文字** - 当不需要展开时

#### 5.2 初始化前准备

```bash

# 2. 检查运行状态
openclaw status

# 3. 查看当前配置
cat ~/.openclaw/config.json | jq '.agents.list'

# 4. 确认工作区目录
ls -la ~/.openclaw/workspace-main/agents/main/
```

**准备个人信息表（6 项信息）**

| 信息项 | 示例 | 用途 |
|--------|------|------|
| 姓名/昵称 | 来合火 | Agent 称呼你 |
| 时区 | Asia/Shanghai | 日程提醒时间 |
| 职业/行业 | 产品营销、内容运营 | 理解专业语境 |
| 核心需求 | 效率提升、内容创作 | 优先优化方向 |
| 沟通偏好 | 简洁直接 | 回答风格 |
| 禁忌事项 | 不要主动发邮件 | 安全边界 |

**重要认知（4 点心理准备）**
- 🤖 OpenClaw 不是通用聊天机器人，是你的**专属工作助理**

- 🔄 需要 1-2 周的磨合期，越用越懂你
- 🔒 所有数据本地存储，隐私安全

1. 点击左侧「**功能**」→「**权限管理**」

### 第十一章：故障排查

#### 11.1 常见问题速查表

| 问题 | 症状 | 解决方案 |
|------|------|---------|
| **失忆症** | 刚说的话就忘 | 检查 memory_search 是否启用 |
| **植物人** | 不理就躺尸 | 检查 HEARTBEAT.md 心跳任务 |
| **胡言乱语** | 一本正经瞎编 | 启用 memory_search + 事实核查 |
| **复读机** | 不停重复 | 翻篇大法 + 清空上下文 |
| **响应慢** | 超过 30 秒 | 检查模型配置，换轻量模型 |
| **误解意图** | 经常理解错 | 补充 USER.md 工作背景 |

#### 11.2 常见错误与解决

| 错误 | 原因 | 解决 | 来源 |
|------|------|------|------|
| 语音发出来是文件 | 格式不对，不是 OPUS | 用 ffmpeg 转 OPUS | 麻小 |
| 模型下载卡住 | GitHub 国内限速 | 换镜像源或代理 | 麻小 |
| 权限不足 | OAuth 未授权 | 引导主人扫码授权 | 麻小 |
| 一次性全抄别人 prompt | 直接不听话 | 边玩边改，逐步迭代 | 董小狐 |
| 群里乱@真人 | 被封禁 | 只@cli_开头的机器人 | 董小狐 |
| 记忆不分层 | 上下文爆炸 | 用 memory/子目录 + 索引 | 董小狐 |
| 不做前置调研 | 返工 + 架构歪 | 先搜本地、再搜网络 | openclaw-li |
| 完成但不验证 | 表面成功，线上翻车 | 必须留证据三件套 | openclaw-li |
| 只修症状不修机制 | 同类 bug 重复 | 每次修复必须补 lesson | openclaw-li |
